# 🏠 DisPlace 插件（简体中文汉化版）

本插件是基于 [MakePlacePlugin](https://github.com/jawslouis/MakePlacePlugin) 修改而来的 Dalamud 插件，用于 **《最终幻想 XIV》（FFXIV）房屋家具布局的自动保存与加载**。

- **原作者**：Drakansoul 👨‍💻
- **汉化与维护**：AlmondCookie 🍪
- **仓库性质**：非官方简体中文汉化与维护版本 🇨🇳

> ⚠️ 本插件最初为作者个人用途开发，作者不保证其始终可用或持续维护。  
> 🔧 本仓库仅对插件界面与文档进行简体中文汉化，不修改任何核心功能逻辑。

---

## ✨ 插件功能简介

DisPlace 插件可以帮助你：

- 📥 自动保存和加载任意房屋的家具布局（室内 / 室外）
- 📋 复制其他房屋的家具布局
- 🔄 从 **MakePlace（已停止维护）** 或 **ReMakePlace** 程序导入 / 导出布局文件
- 🤝 使用与 MakePlace 相同的布局文件格式，兼容性良好

## 📥 安装方法

### 📋 前置条件

- ✅ 已安装并使用 **FFXIV Quick Launcher（XL）** 启动游戏
- ✅ 已启用 Dalamud 插件系统

### 📌 添加插件仓库

1. 🎮 启动游戏并登录角色
2. 💬 在聊天框中输入：
   ```
   /xlsettings
   ```
3. 🔬 打开 **Experimental（实验性）** 标签页
4. 🔧 找到 **Custom Plugin Repositories（自定义插件仓库）**
5. ➕ 添加以下仓库地址（示例）：
   ```
   https://raw.githubusercontent.com/hyirax/DisPlace-cn/master/pluginmaster.json
   ```
6. ✅ 确认该仓库已启用（Enabled）
7. 💾 点击 **Save and Close** 保存并关闭设置

### 🎯 安装插件

1. 💬 在聊天框输入：
   ```
   /xlplugins
   ```
2. 🔍 在插件列表中搜索：
   ```
   DisPlace Plugin（简体中文）
   ```
3. ⬇️ 点击安装并等待完成
4. ▶️ 安装完成后启用插件

## 🎮 基础使用

在聊天框中输入以下命令打开插件界面：

```
/makeplace
```


⚠️ **注意**：

- 🔄 插件的大多数功能 **仅在“旋转家具模式”下可用**
- 🚪 进入方式：
  > 🏠 房屋菜单 → 🪑 室内 / 室外家具 → 🔄 点击「旋转」

---

## ⚙️ 功能说明

### 💾 保存布局（Save / Save As）

- **💾 Save（保存）**  
  将当前房屋布局保存到当前文件中。  
  > 🆕 首次使用插件时该按钮不可用

- **📁 Save As（另存为）**  
  将当前布局保存为新的文件

📌 **提示**：
- 🏠 如果希望将 **室内和室外** 保存到同一个布局文件中：
  - 请分别在室内和室外各执行一次保存

---

### 📂 加载布局（Load / Load From）

- **📂 Load（加载）**  
  从当前布局文件中加载并应用家具布局

- **📁 Load From（从文件加载）**  
  从指定的布局文件加载并应用布局

⚠️ **加载前请确认**：

1. ✅ 已进入 **房屋-布置家具-规划模式下的旋转家具模式**
2. ✅ 勾选了需要应用的楼层
3. ✅ 房屋中已放置布局所需的所有家具：
   - 🎒 插件 **不会** 自动从背包 / 仓库中取出家具
   - 🎨 插件 **不会** 自动为家具染色
   - ⚪ 若重新加载布局，列表中所有家具应显示为 **白色**（而非灰色）

⏳ 加载过程中请耐心等待，家具将依次放置。

---

### ⏱️ 放置间隔（Placement Interval）

- ⚡ 设置每件家具放置之间的时间间隔（毫秒）
- 🐢 数值过低（如 200ms）可能导致放置失败或异常

---

### 🏷️ 标记家具（Label Furniture）

- 👁️ 在游戏画面中为家具显示标签
- 🖱️ 可直接点击标签，将布局中的位置应用到对应家具

## ❓ 常见问题（FAQ）

### ❓ 插件无法正常工作怎么办？

- 🔄 请先确认插件已更新到最新版本
- 🔄 尝试在 **旋转模式** 下使用
- 📢 若问题仍然存在：
  - 🐛 请在本仓库提交 Issue
  - 💬 或前往原作者社区 / Discord 寻求帮助

---

### 🌐 插件是否支持非英文客户端？

**✅ 支持。**  
布局的保存、加载和传输与游戏语言无关。

---

### 📋 是否可以复制其他房屋的布局？

**✅ 可以。**  
与原插件不同，本插件 **不限制** 该功能。

> 💡 信息应当是自由的，可用于学习和研究。

---

### 🎒 插件会使用我背包或仓库里的家具吗？

**❌ 不会。**  
插件只会操作 **已经放置在房屋中的家具**，不会影响你的库存。

---

### 🔒 使用插件是否安全？

⚠️ 插件通常是安全的，但它确实允许一些 **游戏原本不支持的家具放置方式**。

🤔 请自行判断并承担使用风险。

---

### 📏 家具为什么会自动弹回地面或墙面？

这是游戏机制限制导致的：

- 🎈 家具有最小浮空高度限制
- 🧱 墙挂家具存在最小距离限制

📐 如果你使用 MakePlace 程序，可以在设置中启用 **Minimum Float Distance** 以获得可视化提示。

---

### 💥 应用布局时偶尔会崩溃怎么办？

请确保：

- 🖱️ 鼠标未悬停在任何家具上
- 🏃 角色未站在家具放置路径中
- 📦 单个家具上附着的物品数量不超过 10 个

⚠️ 这些问题通常与游戏本身有关，目前无法完全避免。
